API Documentation

URL: /api/wecome
Method: GET
Description: Just to make sure the service is running properly.
Responses:
  • 200 OK - Successfully connected to web service.

URL: /api/supplies
Method: GET
Description: Retrieves all supplies from the database.
Responses:
  • 200 OK - Successfully retrieved list of supplies.
  • 500 Internal Server Error - If there was an error processing the request.

URL: /api/supplies/{id}
Method: GET
Description: Retrieves a specific supply from the database.
Parameters:
  • id (Integer): Required. The identifier of the supply.
Responses:
  • 200 OK - Successfully retrieved the selected supply.
  • 404 Not Found - The requested supply was not found on the source.
  • 500 Internal Server Error - If there was an error processing the request.

URL: /api/supplies
Method: POST
Description: Creates a new supply in the database.
Parameters:
  • name (String): Required. The name of the supply.
  • quantity (Integer): Required. The amount of supply available.
  • price (Float): Required. The price of one unit of supply.
Request Body (JSON): 
{
  "name": "Sample Supply",
  "quantity": 150,
  "price": 25.75
}
Responses:
  • 201 Created - Successfully created new supply.
  • 400 Bad Request - If the request body is incorrect or missing required fields.

URL: /api/supplies/{id}
Method: POST
Description: Updates a selected supply's information in the database.
Parameters:
  • id (URL | Integer): Required. The identifier of the supply.
  • name (String): Required. The name of the supply.
  • quantity (Integer): Required. The amount of supply available.
  • price (Float): Required. The price of one unit of supply.
Request Body (JSON): 
{
  "name": "Edited Supply",
  "quantity": 300,
  "price": 99.98
}
Responses:
  • 201 Created - Successfully updated supply information.
  • 400 Bad Request - If the request body is incorrect or missing required fields.

URL: /api/supplies/{id}
Method: DELETE
Description: Deletes a specific supply from the database.
Parameters:
  • id (Integer): Required. The identifier of the supply.
Responses:
  • 200 OK - Successfully deleted the selected supply.
  • 404 Not Found - The requested supply was not found on the source.
  • 500 Internal Server Error - If there was an error processing the request.
